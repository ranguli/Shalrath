# Shalrath

## Building

### Dependencies

```sh
sudo apt install cmake qt6-base-dev qt6-base-dev-tools qt6-tools-dev libgl1-mesa-dev libglu1-mesa-dev clang-tidy clang-format cppcheck
```

```sh
mkdir build && cd build
cmake ..
make
```
